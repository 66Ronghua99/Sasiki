# PROJECT_LOGS

## Purpose
- `PROJECT_LOGS.md` 是 append-only 项目流水账。
- 记录每一次重要决策、尝试、推翻、方向调整、验证结果和文档职责变化。
- 默认不进入当前任务执行的读取序列；只有在项目级 inspect、复盘、方向追溯或历史决策争议时按需读取。
- 除非人工明确触发清理，否则只追加，不折叠、不重写历史。
