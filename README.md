# Sasiki

> 观察用户操作，自动生成可复用的工作流 Agent

Sasiki（日语「摹す」- 临摹、模仿）通过录制你的屏幕操作，使用多模态 AI 理解你的工作流程，并生成可重复使用的自动化工作流。

## 核心理念

**你演示一遍，AI 学会你的工作方式**

不需要写代码，不需要描述需求，只需正常工作，Sasiki 会自动学习并为你生成可复用的工作流。

## 快速开始

```bash
# 安装
pip install -e ".[dev]"

# 配置 API Key
cp .env.example .env
# 编辑 .env，添加 OPENROUTER_API_KEY

# 录制你的工作流
sasiki record
# 正常工作...
# Ctrl+C 停止录制

# 分析生成工作流
sasiki analyze ~/.sasiki/recordings/20240115_143022_xxxxx

# 查看已保存的工作流
sasiki list

# 查看工作流详情
sasiki show "法律合同起草"
```

## 使用场景

- **法律合同起草**: 搜索法条 → 整理要点 → 生成合同
- **竞品价格监控**: 访问网站 → 提取数据 → 更新表格
- **周报生成**: 提取项目数据 → 整理 → 生成报告
- **任何重复性桌面工作**

## 架构

```
录制 (Screen) → 分析 (VLM) → 提取 (Workflow) → 复用 (Replay)
```

1. **录制层**: 屏幕录制 + 事件捕获
2. **分析层**: 多模态 LLM 理解操作序列
3. **工作流层**: 结构化工作流定义
4. **执行层**: 基于视觉的回放（开发中）

## 成本

分析阶段使用多模态 LLM，但仅录制时产生成本：

- 1小时操作约 600-1200 帧
- 成本约 $2-4/小时
- 录制一次，永久复用
